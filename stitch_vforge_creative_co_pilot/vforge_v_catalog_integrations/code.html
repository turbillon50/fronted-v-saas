Technical
Glossary

Reference documentation for absolute system control. Access endpoints, context protocols, and CLI references.

vforge-sys-docs
chevron_right DEFINE: API (Application Programming Interface)

The primary interface boundary for vForge modules. Utilizes strict RESTful conventions with predictable resource-oriented URLs, returns JSON-encoded responses, and uses standard HTTP response codes. Authentication requires bearer tokens generated via the Auth Protocol module.

chevron_right DEFINE: MCP (Model Context Protocol)

An open standard enabling secure, two-way connections between data sources and AI agents. vForge implements MCP to allow the Neural Core to seamlessly ingest context from the Data Node (Neon) without compromising isolation layers or requiring custom ingestion pipelines.

chevron_right DEFINE: Terminal (vForge CLI)

The headless control plane. A lightweight binary vf-core allowing architects to provision, wire, and tear down marketplace modules via declarative YAML configurations. Prioritizes silent execution over verbose output.
